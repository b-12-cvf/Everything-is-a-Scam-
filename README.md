The EIAS ecosystem operates on a **Verify-then-Publish** model:
1.  **Report:** Users submit evidence (Tx hashes, screenshots, logs) via Discord.
2.  **Verify:** Moderators & Community Analysts validate the claim against blockchain data.
3.  **Publish:** Verified bad actors are added to the Global Blacklist.
